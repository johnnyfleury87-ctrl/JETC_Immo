# Branding Assets

## Structure des dossiers

```
/public/branding/
├── jetc/
│   ├── logo.png (logo JETC IMMO - fond clair)
│   └── logo-white.png (logo JETC IMMO - fond foncé)
└── perriti/
    ├── logo.png (logo Perriti - fond clair)
    └── logo-white.png (logo Perriti - fond foncé)
```

## Instructions

Placez les fichiers PNG des logos dans les dossiers correspondants.
Les images sont référencées dans le code via `/branding/jetc/logo.png` etc.
